# Gemini Bakend

Este backen es para pruebas con la api de Gemini, usando Flutter como frontend.

## Dev

```bash
$ npm install
```
- Crear el archivo `.env` basado en el `.env.template`


## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
